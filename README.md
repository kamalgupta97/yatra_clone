# yatra_clone


We have cloned e-commerce website Yatra. Yatra.com is an Indian online travel agency and travel search engine.

    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Bhargavi952/yatra_clone.git
```

Go to the project directory

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
  
## Dependencies

 - [Material UI](https://material-ui.com/getting-started/installation/)
 - [Material-UI Icons](https://material-ui.com/components/icons/#icons)
 - [React Icons](https://react-icons.github.io/react-icons/)
 
  
## Tech Stack

- Client: React, Redux 

- Server:  Node, Express

- DataBase: MongoDB

  
## Features

- Authentication
- Bus selection
- Seat selection
- Payment Gateway (Stripe)
- User information
